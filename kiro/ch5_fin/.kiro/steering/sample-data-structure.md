# sample_data.xlsx 構造定義

このプロジェクトで使用する `sample_data.xlsx` の構造情報。

## シート一覧と役割

| シート名           | 役割                                                                   |
| ------------------ | ---------------------------------------------------------------------- |
| 設備マスタ         | 工場の設備（機械）のマスタ情報。設備の基本属性を管理                   |
| ステータス変更履歴 | 各設備の稼働状態の変遷を時系列で記録したイベントログ                   |
| センサーデータ     | 各設備に取り付けられたセンサーの計測値を10分間隔で記録した時系列データ |

## 各シートのカラム構成

### 設備マスタ（8件）

| カラム名 | データ型       | サンプル値                                           |
| -------- | -------------- | ---------------------------------------------------- |
| 設備名   | 文字列         | `CNC旋盤 A-01`, `プレス機 B-01`, `溶接ロボット D-01` |
| タイプ   | 文字列（4種）  | `CNC旋盤`, `プレス機`, `射出成形機`, `溶接ロボット`  |
| 設置場所 | 文字列（4種）  | `A棟1F`, `A棟2F`, `B棟1F`, `B棟2F`                   |
| 設置日   | 文字列（日付） | `2020-04-15` 〜 `2023-02-14`                         |

- 設備IDカラムは存在しない。行番号（0始まり）+ 1 が暗黙のIDとして他シートから参照される。

### ステータス変更履歴（59件）

| カラム名         | データ型          | サンプル値                                                          |
| ---------------- | ----------------- | ------------------------------------------------------------------- |
| 設備ID           | 整数（1〜8）      | 設備マスタの行番号+1に対応                                          |
| 設備名           | 文字列            | 設備マスタの設備名と一致（非正規化）                                |
| 発生日時         | 文字列（ISO8601） | `2026-03-01T10:12:17` 〜 `2026-03-08T03:00:17`                      |
| 変更前ステータス | 文字列（4種）     | `稼働中`, `異常`, `メンテナンス中`, `停止中`                        |
| 変更後ステータス | 文字列（4種）     | 同上                                                                |
| 理由             | 文字列            | `冷却水温度が上限値を超過`, `ベアリング摩耗による振動増加を検知` 等 |

### センサーデータ（1,152件）

| カラム名       | データ型          | 範囲                                           | 備考                                        |
| -------------- | ----------------- | ---------------------------------------------- | ------------------------------------------- |
| 設備ID         | 整数（1〜8）      | -                                              | 設備マスタの行番号+1に対応                  |
| タイムスタンプ | 文字列（ISO8601） | `2026-03-07T10:22:17` 〜 `2026-03-08T10:12:17` | 約24時間分                                  |
| temperature    | 浮動小数          | 0.0 〜 293.7                                   | 全設備で記録あり                            |
| vibration      | 浮動小数          | 0.0 〜 7.27                                    | 全設備で記録あり                            |
| rpm            | 浮動小数          | 0.0 〜 2502.66                                 | CNC旋盤のみ（設備1,2）。プレス機4は一部あり |
| power_kw       | 浮動小数          | 0.0 〜 44.65                                   | 全設備で記録あり                            |
| pressure       | 浮動小数          | 0.0 〜 138.73                                  | プレス機・射出成形機（設備3〜6）のみ        |

## シート間の関連性

- 設備マスタの「行インデックス + 1」が、他2シートの `設備ID` に対応
  - 1=CNC旋盤 A-01, 2=CNC旋盤 A-02, 3=プレス機 B-01, 4=プレス機 B-02
  - 5=射出成形機 C-01, 6=射出成形機 C-02, 7=溶接ロボット D-01, 8=溶接ロボット D-02
- ステータス変更履歴には `設備名` が非正規化で含まれており、設備マスタと完全一致
- センサーデータの `rpm` と `pressure` のNULL有無は設備タイプに依存:
  - CNC旋盤（ID 1,2）: rpm あり / pressure なし
  - プレス機（ID 3,4）: pressure あり / rpm は基本なし（ID 4のみ一部あり）
  - 射出成形機（ID 5,6）: pressure あり / rpm なし
  - 溶接ロボット（ID 7,8）: rpm・pressure ともに基本なし（ID 8は一部あり）

## データの件数・傾向

- 設備マスタ: 8台。各タイプ2台ずつ、4拠点に分散配置
- ステータス変更履歴: 59件、期間は2026-03-01〜03-08の約1週間。CNC旋盤 A-01が最多（11件）で頻繁に状態遷移。最頻の変更前ステータスは「稼働中」（24件）、変更後は「稼働中」（21件）で、異常→メンテナンス→復帰のサイクルが多い
- センサーデータ: 1,152件（8設備 × 144レコード）。10分間隔で約24時間分。停止中の設備（ID 8末尾など）はtemperature=20.0, vibration=0.1のような低値・一定値になっており、稼働状態との相関がある
