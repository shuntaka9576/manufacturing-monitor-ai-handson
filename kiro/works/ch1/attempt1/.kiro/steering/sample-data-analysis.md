# sample_data.xlsx 構造分析

## シート一覧と役割

| シート名 | 役割 | 件数 |
|---|---|---|
| 設備マスタ | 工場設備の基本情報 | 8件 |
| ステータス変更履歴 | 各設備の稼働状態の変遷ログ | 59件 |
| センサーデータ | 各設備のセンサー計測値（時系列） | 1152件 |

## 各シートのカラム構成

### 設備マスタ（8行 × 4列）

| 列名 | データ型 | サンプル値 |
|---|---|---|
| 設備名 | 文字列 | CNC旋盤 A-01 |
| タイプ | 文字列 | CNC旋盤, プレス機, 射出成形機, 溶接ロボット |
| 設置場所 | 文字列 | A棟1F, B棟1F, A棟2F, B棟2F |
| 設置日 | 文字列(日付) | 2020-04-15 |

※ 設備IDの列がなく、行番号（1〜8）が暗黙のIDとなっている。

### ステータス変更履歴（59行 × 6列）

| 列名 | データ型 | サンプル値 |
|---|---|---|
| 設備ID | 整数 | 1〜8 |
| 設備名 | 文字列 | CNC旋盤 A-01 |
| 発生日時 | 文字列(ISO8601) | 2026-03-01T10:12:17 |
| 変更前ステータス | 文字列 | 稼働中, 異常, メンテナンス中, 停止中 |
| 変更後ステータス | 文字列 | 同上 |
| 理由 | 文字列 | 冷却水温度が上限値を超過 |

### センサーデータ（1152行 × 7列）

| 列名 | データ型 | 備考 |
|---|---|---|
| 設備ID | 整数 | 1〜8 |
| タイムスタンプ | 文字列(ISO8601) | 10分間隔 |
| temperature | 浮動小数点 | 温度(℃) |
| vibration | 浮動小数点 | 振動値 |
| rpm | 浮動小数点 or None | 回転数（CNC旋盤のみ） |
| power_kw | 浮動小数点 | 消費電力(kW) |
| pressure | 浮動小数点 or None | 圧力（プレス機・射出成形機のみ） |

## シート間の関連性

- 設備マスタの行番号（1〜8）= 他2シートの「設備ID」
- ステータス変更履歴には設備名も冗長に保持しており、設備マスタと一致
- センサーデータも設備IDで設備マスタと紐づく

```
設備マスタ (行番号=暗黙ID)
  ├── ステータス変更履歴.設備ID
  └── センサーデータ.設備ID
```

## データの傾向

- センサーデータは全設備均等に144件ずつ（10分間隔 × 約24時間分）
- 設備タイプごとにセンサー値の特性が異なる:

| タイプ | 温度帯 | rpm | pressure | 特徴 |
|---|---|---|---|---|
| CNC旋盤 (ID 1,2) | 29〜53℃ | あり(平均1700〜1800) | なし | rpmが主要指標 |
| プレス機 (ID 3,4) | 28〜46℃ | なし | あり(14〜20MPa) | 圧力が主要指標。ID4は停止期間あり(0値混在) |
| 射出成形機 (ID 5,6) | 185〜294℃ | なし | あり(63〜139) | 高温・高圧。ID6に温度異常(293℃)あり |
| 溶接ロボット (ID 7,8) | 41〜68℃ | なし | なし | ID8は停止期間あり(0値混在) |

- ステータス変更履歴は2026-03-01〜03-08の1週間分
- 全8台に何らかのイベントが発生
- 典型パターン: 稼働中 → 異常 → メンテナンス中 → 稼働中
