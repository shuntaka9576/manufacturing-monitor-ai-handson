# 意味類似検索のサンプルクエリと期待結果

semantic search ページの検証に使用する。キーワード一致ではなく意味的類似性で結果が返ることがポイント。
PLaMo Embedding (`pfnet/plamo-embedding-1b`) + cosine similarity、閾値 0.55 での実測値に基づく。

| クエリ                     | 1位ヒット (類似度)                          | 検証観点                                  |
| -------------------------- | ------------------------------------------- | ----------------------------------------- |
| 油圧の圧力が下がった       | 油圧圧力が規定値を下回る低下を検知 (74.0%)  | 「下がった」→「規定値を下回る低下」       |
| 冷却水の温度が高すぎる     | 冷却水温度が上限値を超過 (71.4%)            | 「高すぎる」→「上限値を超過」             |
| 金型交換                   | 新規金型セットアップのため停止 (70.7%)      | 「交換」→「セットアップ」                 |
| 材料不足                   | 材料補充のため一時停止 (65.1%)              | 「不足」→「補充」（対義語の意味的紐付け） |
| 温度が異常                 | 温度異常上昇を検知 (65.2%)                  | 「異常」→「異常上昇」                     |
| 定期点検                   | 年次法定点検のため計画停止 (64.5%)          | 「定期」→「年次法定」                     |
| ロボットの位置がずれている | ロボットアーム位置ずれを検知 (64.3%)        | 「位置がずれている」→「位置ずれを検知」   |
| 精度が悪い                 | 加工精度低下により調整停止 (59.2%, 3位)     | 「悪い」→「低下」                         |
| 品質に問題がある           | 原材料品質不良によるロット停止 (58.9%, 3位) | 「問題がある」→「不良」                   |
