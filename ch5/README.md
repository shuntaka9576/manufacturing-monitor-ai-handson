# ch5: PLaMo Embedding - 意味類似検索

## 概要

PLaMo Embedding（`pfnet/plamo-embedding-1b`）を使い、ステータス変更履歴を自然言語で意味類似検索できるページを追加する。
キーワード一致ではなく、意味的な類似性に基づいて関連する履歴を検索できる。

## 体験すること（約3分｜経過 約3分）

Kiroを使い、PLaMo Embeddingモデルによる意味類似検索機能の実装を体験します。
テキストをベクトル化し、コサイン類似度で検索するという自然言語処理の基本的な手法を、実際のアプリケーションに組み込む流れを学びます。

### Embeddingとは

Embedding（埋め込み表現）とは、テキストを固定長の数値ベクトルに変換する技術です。意味が近いテキスト同士はベクトル空間上で近い位置に配置されるため、コサイン類似度を計算することで「意味的に似ている」テキストを検索できます。

キーワード検索では「振動の異常」と「ベアリング摩耗による振動増加」は一致しませんが、Embeddingを使った意味類似検索では高い類似度で検出できます。
